⚠️ SI ALGO FALLA
═══════════════════════════════════════════════════════════════════════════════

La razón más común (95% de los casos):
  → Files missing in versión 1.3/extension/

Solución rápida:
  1. cp -r installer/dist/extension/* "versión 1.3/extension/"
  2. git add . && git commit -m "Extension files" && git push
  3. Github Actions se compila nuevamente

Para otros errores:
  → Lee GITHUB_ACTIONS_SETUP.md sección "Troubleshooting"
  → Revisa logs del workflow en GitHub
  → Busca "Cannot find setup.iss" si no ve el archivo
